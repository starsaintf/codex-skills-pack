# Notices

This skill pack includes skills copied from `anthropics/skills`.

The imported Anthropic example skills include their own `LICENSE.txt` files and are distributed under the license terms in those folders. Keep those license files with the corresponding skill directories when redistributing.

The restricted/source-available Anthropic document skills (`docx`, `pdf`, `pptx`, `xlsx`) are intentionally not included in this repository.

Codex system skills from `~/.codex/skills/.system` are intentionally not included.

This skill pack also includes skills copied from the local OpenAI-curated Vercel plugin cache for `vercel/vercel-plugin` version `0.21.0`. The plugin metadata identifies that upstream package as Apache-2.0. Vercel-derived skill directories include Apache-2.0 `LICENSE.txt` files so individually installed skills carry the license text.

Additional installable skills were promoted from local plugin-backup temp folders after checking available plugin metadata. Some promoted skills contain their own `LICENSE.txt`; those embedded licenses take precedence over older metadata summaries. See `manifest.json` and `SKILLS.md` for per-skill source, license, evidence, origin, and audit-status metadata.

The proprietary OpenAI Developers temp plugin and large Twilio temp plugin were not included in this promotion batch.
